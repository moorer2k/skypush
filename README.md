# skypush
![skypush](https://cloud.githubusercontent.com/assets/2856413/15801357/060736fe-2a47-11e6-8d5c-b0a544150886.png)

Push to talk for skype!

A small program to enable you operate Skype as if it was a Push To Talk based client, like TeamSpeak etc. 


# NOTICE
You must accept the request to attach to your Skype client before this will function. 

![skype](https://cloud.githubusercontent.com/assets/2856413/15801358/0715daa0-2a47-11e6-87fd-7e08852b5625.png)

# CREDITS
For the hotkey functionality I am using [ManagedWinAPI](https://github.com/mwinapi/mwinapi)
